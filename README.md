# iPhone 上，自動化 UITest 簡單範例

早期一開始就有簡單研究 UI 自動化測試，大多數會需要他都是為了使用腳本
自動化來測試 app 流程是否有哪一部份被改壞，而達到減少人工測試的成本， 不同於其他目的，我當初學這個目的反而更單純，
想要寫一份腳本，幫助我操作某些遊戲 app，哈，雖然後來發現不能控制別人的 app = =，
目前實作比較偏寫自動化流程，而檢查，判斷就比較少寫
